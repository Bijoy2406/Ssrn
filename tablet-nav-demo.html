Tablet Navigation Component

This is the tablet-optimized navigation bar for SSRN. It features horizontal layout with dropdown menus for "Our Services" and "For Partners", along with direct links for "About Us" and "Help".

Component Features:

  • SSRN Logo - 137px × 48px branding element
  • Dropdown Navigation - "Our Services" and "For Partners" with chevron indicators
  • Direct Links - "About Us" and "Help" for quick access
  • Sign In Button - Outlined button with user icon
  • Hover States - Subtle background color changes on interaction
  • Smooth Animations - Dropdown fade-in and chevron rotation

Design Specifications:

  • Navigation bar padding: 16px vertical, 48px horizontal
  • Items gap: 8px between navigation items
  • Dropdown appears 8px below button
  • Dropdown min-width: 229px
  • Box shadow: 0 0 5px rgba(3, 17, 13, 0.1)
  • Border radius: 8px (small) and 12px (medium)

Interactions:

  • Larger Tablets (>1024px): Horizontal navigation with dropdowns
  • Smaller Tablets (≤1024px): Hamburger menu opens sidebar
  • Click dropdown buttons to reveal menu options
  • Click outside dropdown/sidebar to close
  • Hover over items for visual feedback
  • Chevron rotates 180° when dropdown is open
  • Press Escape key to close sidebar

Responsive Behavior:

  • Desktop (>1024px): Full horizontal navigation with all items visible
  • Tablet (≤1024px): Logo + Hamburger menu → Opens sidebar overlay
  • Sidebar: 272px width, slides in from left, gray overlay backdrop
  • Sidebar Content: Sign In button, collapsible dropdowns, Help link

Note: Resize your browser window to see the responsive behavior in action! The navigation automatically switches between horizontal layout and sidebar menu based on screen width.
