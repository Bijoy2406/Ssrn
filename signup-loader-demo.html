This demo shows the signup email loader (skeleton screen) that displays while the signup or email verification page is loading. The skeleton loader provides visual feedback to users that content is being fetched, improving perceived performance.

Features:

  • Skeleton placeholders matching signup form layout
  • Smooth shimmer animation effect
  • Responsive design that adapts to all screen sizes
  • Accessible with screen reader support
  • Respects user's motion preferences (prefers-reduced-motion)
  • Smooth fade-out transition when content loads
  • Centered content layout at 520px width
