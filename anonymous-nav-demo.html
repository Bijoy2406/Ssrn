Anonymous User Navigation

This navigation bar is designed for anonymous (non-authenticated) users landing on the SSRN website. It features dropdown menus for "Our Services" and "For Partners", simple links for "About Us" and "Help", and a prominent "Sign In" button on the right.

The navigation is fully responsive and includes a mobile menu toggle for smaller screens. Hover over the dropdown menu items to see the hover states, and click on the navigation items to toggle the dropdown menus.

Features:
• Sticky navigation bar with drop shadow
• Dropdown menus with icons and lock indicators
• Responsive design with mobile menu
• Smooth animations and transitions
• Accessibility support with ARIA attributes
• Uses design tokens from tokens.css
