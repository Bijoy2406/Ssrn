Booking Form Component

This is the booking form component for the anonymous user landing page. It allows users to search for rides by entering their pickup/dropoff locations, date, and time.

Features

  • Toggle between "One Way Ride" and "Hourly Ride"
  • Location input fields with icons
  • Date picker with calendar icon
  • Time picker with clock icon
  • Search button to see prices
  • Fully responsive design
  • Form validation
  • Loading states

Form Fields

  • From: Pickup location (airport, address, hotel)
  • To: Dropoff location (airport, address, hotel)
  • Date: Ride date selection
  • Pickup Time: Time selection

Try it!

Fill out the form on the right and click "See Prices" to simulate a booking search. Toggle between ride types to see how the form adapts.
